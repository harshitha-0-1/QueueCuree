# QueueCure 🏥

A real-time hospital queue management system.

## Features

- Patient can join queue
- Automatic token generation
- Reception dashboard
- Call next patient
- Live waiting room updates
- Queue clearing option
- Real-time communication using Socket.IO

## Tech Stack

Frontend:
- React.js
- Vite
- CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Real-time:
- Socket.IO
